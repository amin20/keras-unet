## Installation
To install, run:
```bash
make install
```

## Building the Model
Import packages for building the model:
```python
from models.unet import build_model
```
Build the network with desired parameters:
```python
m = build_model((3, 3), [64, 128, 256, 512], dropout=0.1, padding='same')
```
The summary of the model shows the layers & parameters of the models:
```python
m.summary()
```
